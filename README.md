# BankSys

### Requirements

[Node.js](https://nodejs.org/en/)

[MongoDB](https://www.mongodb.com/)

### Quick Start

#### Clone
```
git clone https://github.com/firstnewline/BankSys.git
```
#### Start
```
cd BankSys
npm install
npm start
```
then vist http://localhost:3000
