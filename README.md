# BankSys
homework

### Requirements

Node.js

Mysql

Redis

### Quick Start

#### Clone
```
git clone https://github.com/firstnewline/BankSys.git
```
#### Start
connect to your mysql and redis
```
cd BankSys
npm install
npm start
```
then vist http://localhost:3000
